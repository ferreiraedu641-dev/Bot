📌 Funções

- Sistema de apostas com botões
- Criação automática de salas privadas
- Anti-spam
- Logs

---

⚙️ Instalação

1. Instale Python 3.10+

2. Instale dependências:
   pip install -r requirements.txt

3. Coloque seu TOKEN:

- Crie um bot no Discord Developer Portal
- Copie o TOKEN
- Defina como variável de ambiente:
  TOKEN=seu_token_aqui

4. Rode o bot:
   python main.py

---

🛠️ Comandos

!apostado → cria partida
!setlogo → define logo
!setbanner → define banner
!setcanal → define canal
!setcategoria → define categoria
!setlogs → define logs
!setantispam → ativa anti-spam
!configurar → ver configs
